Power BI Embedded Sample

This sample is developed using NodeJS Framework.
In order to see a different report, make changes to the config.json file.

Code is present in the following files:

  1. embedConfigService.js
  2. authentication.js
  3. server.js
  4. index.js
  5. utils.js
  6. index.html
  7. index.css

For Live demo and more code samples please visit https://aka.ms/pbijs
For JavaScript API, please visit https://aka.ms/PowerBIjs
