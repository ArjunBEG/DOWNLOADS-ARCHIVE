   ## [BEM](http://getbem.com/naming/) - Block, Element, Modifier

    1. Block
    ----------------------------------------------

    Eg: menu

    <ul class="menu"></ul>

    2. Elements
    ----------------------------------------------

    Eg: menu__list

    <ul class="menu">
      <li class="menu__list"></li>
    </ul>

    3. Modifiers
    ----------------------------------------------

    Eg: menu__list--active

    <ul class="menu">
      <li class="menu__list"></li>
      <li class="menu__list menu__list--active"></li>
    </ul>
