Background

Web Audio JavaScript syntax can be shortened and simplified with method chaining for the AudioNode.connect() and AudioParam.connect() automation methods.

This sample does the following:

  1. Creates an AudioContext.
  2. Creates an audio source node.
  3. Creates a gain node.
  4. Creates a filter node.
  5. Connects nodes from the audio source to the AudioContext destination.
  6. Plays sound!
