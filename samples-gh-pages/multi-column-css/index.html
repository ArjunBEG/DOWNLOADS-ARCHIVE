Multi-column CSS Sample

Available in Chrome 44+

While Chrome has supported CSS styles related to multi-column layouts for some time now, Chrome 44 includes a rewritten implementation that resolves longstanding bugs and brings support up to par with what other browsers offer.

The following example shows a three column layout, where each column has a width of 10em and with gaps of 2em between columns. If the viewport is not wide enough to fit three columns, the browser will automatically use fewer columns.

As of Chrome 44, the -webkit- prefix is still required when using column layout-related styles.
