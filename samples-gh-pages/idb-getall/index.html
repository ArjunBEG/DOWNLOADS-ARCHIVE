Background

The IndexedDB "batched get" API allows you to retrieve several keys or values from IndexedDB in a single method call. The "batched get" methods include:

  • IDBObjectStore.getAll()
  • IDBObjectStore.getAllKeys()
  • IDBIndex.getAll()
  • IDBIndex.getAllKeys()
